# Simoleon Bank Converter

Next.js app for converting world currencies to Simoleon and managing a Sims-inspired local player bank ledger.

## Features
- World currency conversion (including IDR)
- Simoleon balance tracking
- Income/expense entries
- Paginated transaction table (bank mutation style)
- LocalStorage persistence
- Sims-inspired UI palette

## Run
```bash
npm install
npm run dev
```

## Deploy
Ready for Vercel deployment.
